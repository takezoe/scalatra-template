scalatra-template
=================

Template project for Scalatra based web application.

##Usage

Clone this repository and overwrite organization and name in `build.sbt` and hit the command below:

```
./sbt ~jetty:start
```

##Components

* Scala 2.12.0
* scalatra 2.5.0
* twirl 1.3.0
* slick 3.2.0-M2
* scalatra-forms 1.1.0
* sbt 0.13.13

scalatra-template
=================

Template project for Scalatra, Twirl and Slick based web applications.

## Usage

You can create and run a project as following:

```
$ sbt new takezoe/scalatra-twirl-slick
$ cd <name-of-app>
$ sbt ~jetty:start
```

## Components

* Scala 2.12.1
* scalatra 2.5.0
* twirl 1.3.0
* slick 3.2.0-M2
* scalatra-forms 1.1.0
